# Descricao para portfolio (GitHub / LinkedIn) + uso comercial

## Short version (EN)

Mini Commercial CRM built with HTML, CSS, and vanilla JavaScript, focused on client registration, search, and follow-up. The project demonstrates local data handling with `localStorage`, responsive UI, and UX improvements for a simple but practical business workflow.

## Technical highlights (EN)

- Demo authentication flow;
- Client registration and lookup;
- Multi-field search;
- Legacy-data normalization in local storage;
- Code structure ready to evolve to a backend architecture.

## How to present this project (EN)

This project can be presented as an example of:

- functional prototype for a commercial workflow;
- front-end organization for an MVP;
- solid base to evolve into a CRM with API and database.

## Complementary commercial positioning (EN)

Although this is a portfolio project, it can also be referenced in commercial conversations
to demonstrate ability to:

- build customized CRM solutions;
- organize client registration and relationship flow;
- deliver a functional MVP focused on business outcomes.
